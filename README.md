# Map Page Challenge

## Challenge:
Build a map page resembling Google Maps using HTML and CSS.

## Requirements:
Utilize the features of the selected CSS framework to develop a responsive and visually appealing map page. Customize the design to resemble the layout and functionality of Google Maps as closely as possible.
